# ML & AI Fundamentals

Полное руководство по основам машинного обучения и искусственного интеллекта. Фундаментальные знания, которые останутся актуальными независимо от изменений в фреймворках и библиотеках.

## 📚 Содержание

- **Основы ML** - типы обучения, метрики, кросс-валидация
- **Нейронные сети** - архитектуры, функции активации, оптимизаторы
- **Глубокое обучение** - CNN, RNN, Transformers, Attention mechanisms
- **Обработка данных** - feature engineering, нормализация, аугментация
- **Моделирование** - bias-variance tradeoff, регуляризация, ансамбли
- **Этика и безопасность** - fairness, interpretability, adversarial attacks

## 🚀 Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/DaniilGavrin/ml-ai-fundamentals.git

# Установка зависимостей для локальной сборки
pip install mkdocs mkdocs-material

# Запуск локального сервера
mkdocs serve
```
